"자기소개서를 위한 텍스트 편집기"는 py-hanspell이라는 네이버 맞춤법 검사기를 이용한 파이썬용 한글 맞춤법 검사 라이브러리를 이용한 맞춤법 검사 프로그램입니다. 해당 라이브러리와 관련하여 자세한 설명이 필요하시다면 https://github.com/ssut/py-hanspell를 참고해주세요.

▶ 프로젝트 안에 py-hanspell이 설치되어 있으니 requests 라이브러리만 설치하시면 됩니다.





▶ resquests 라이브러리를 설치하는 방법은

1. 터미널에서 pip는 파이썬이 설치된 폴더 하위에 Scripts 폴더에 있으므로 Scripts 폴더까지 이동합니다.
2. pip install requests 라고 입력하여 설치를 진행합니다.
3. 설치를 완료합니다.

저는 VSCode를 사용해 VSCode 터미널에  바로 pip install requests를 입력하여 설치했습니다.





▶ 해당 프로젝트 파일의 Main.py에서 파이썬 파일 실행하면 정상적으로 실행됩니다.




▶ 글자 수 세기 기능은 구현하지 못했고 수정사항이 많지만 3.Design 문서는 수정하지 못했습니다.





▶ 사용 방법

1. 실행하면 첫 화면의 좌측 텍스트박스에 글을 입력할 수 있습니다.
2. 양 텍스트박스 사이의 맞춤법검사 버튼을 누를 시 좌측 텍스트 박스에 입력된 글에 대한 맞춤법 검사 결과가 우측 텍스트 박스에 출력됩니다.
3. 텍스트박스에 대해 서식을 지정할 수 있습니다. 좌측 상단의 서식 버튼을 누를 경우 서식을 지정할 수 있는 창이 나타납니다. 글꼴모양, 글꼴크기, 글꼴 스타일, 서식 적용할 텍스트박스를 지정할 수 있습니다. 글꼴 크기를 크게 설정할 경우 창이 지나치게 커질 수 있으니 참고해주세요.
4. 우측 하단에 저장, 불러오기 버튼이 있습니다. 해당 기능은 로그인을 해야 이용 가능합니다. 저장을 누를 시 좌측 텍스트 상자에 입력된 글이 저장됩니다. 불러오기를 누를 시 죄측 텍스트 상자에 이전에 저장된 글이 출력됩니다.
5. 우측 상단에 로그인 버튼과 회원가입 버튼이 있습니다. 회원가입을 한 적이 없으시면 먼저 회원가입을 진행해주세요. 회원가입을 하셨다면 정상적으로 로그인할 수 있습니다.
6. 좌측 하단에 모두 지우기 버튼이 있습니다. 좌측에 텍스트박스에 입력된 글을 지울 수 있습니다. 되돌릴 수 없으니 주의해주세요.
